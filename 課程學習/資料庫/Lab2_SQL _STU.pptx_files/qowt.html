GROUP BY

GROUP BY 敘述句搭配 (aggregation function) 使用，是用來將查詢結果中特定欄位值相同的資料分為若干個群組，而每一個群組都會傳回一個資料列。若沒有使用 GROUP BY，聚合函數針對一個 SELECT 查詢，只會返回一個彙總值。

aggregation function指的也就是 AVG()、COUNT()、MAX()、MIN()、SUM() 等這些內建函數。

38

GROUP BY

SELECT column_name(s), aggregate_function(column_name) FROM table_name WHERE column_name operator value GROUP BY column_name1, column_name2...;

Alias(AS)

在 SQL 中我們可以替資料表或欄位名稱取一個別名 (Alias)，這可以使名稱複雜的 SQL 查詢語句更易讀且可以有更直觀的查詢結果。

SELECT table_column AS alias_name FROM table_name;

這樣顯示出來的畫面，就是table_column名稱變成alias_name

Having

HAVING 子句是用來取代 WHERE 搭配聚合函數 (aggregate function) 進行條件查詢，因為 WHERE 不能與聚合函數一起使用。

就是Group by版本的where條件^^

49

Having

SELECT column_name(s), aggregate_function(column_name) FROM table_name WHERE column_name operator value GROUP BY column_name1, column_name2... HAVING aggregate_function(column_name) operator value;
